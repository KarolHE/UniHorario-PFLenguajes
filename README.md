# UniHorario – Sistema inteligente de planificación de horarios universitarios

---

## Introducción

La planificación de horarios universitarios es uno de los problemas más complejos en la gestión académica. Asignar cursos, docentes y aulas de manera eficiente — respetando restricciones, preferencias y recursos disponibles — es una tarea que tradicionalmente se realiza de forma manual, consumiendo tiempo y generando errores difíciles de detectar.

**UniHorario** es un sistema multilenguaje que automatiza este proceso de forma inteligente. Combina el poder del razonamiento lógico de Prolog, la solidez de los modelos de datos en Scala y la flexibilidad de Python para orquestar, optimizar y aprender de cada ejecución. El resultado es un sistema que no solo genera horarios válidos, sino que mejora con el tiempo, explica sus decisiones y equilibra múltiples objetivos a la vez.

---

## Objetivos

### Objetivo general

Desarrollar un sistema inteligente multilenguaje que permita generar y organizar horarios universitarios de manera eficiente, reduciendo conflictos, optimizando la distribución de recursos académicos y adaptándose a las condiciones cambiantes del entorno universitario.

### Objetivos específicos

- Gestionar la información de cursos, docentes y aulas mediante modelos de datos robustos y tipados en Scala.
- Aplicar restricciones académicas complejas para la generación de horarios válidos utilizando razonamiento lógico en Prolog.
- Detectar y explicar conflictos de horarios de forma automática, indicando la causa exacta de cada incompatibilidad.
- Generar propuestas de horarios optimizadas considerando simultáneamente la carga del docente, la disponibilidad de aulas y las preferencias académicas.
- Incorporar memoria adaptativa que registre configuraciones fallidas y las evite en futuras generaciones.
- Facilitar la planificación académica universitaria mediante una interfaz clara que presente resultados y alternativas de forma comprensible.
